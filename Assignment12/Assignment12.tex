\section{\textbf{Problem}}
Let $\vec{N}$ be a 2 Ã— 2 complex matrix such that $\vec{N}^2$ = 0. Prove that either $\vec{N}$ = 0 or $\vec{N}$ is similar over $\mathbb{C}$ to\\
\begin{align}
\myvec{0&0\\1&0}
\end{align}
\section{\textbf{Solution}}
\begin{align}
\mbox{Let }\vec{N}&=\myvec{a&b\\c&d}\\
\mbox{Since }\vec{N}^2&=0
\end{align}
If $\myvec{a\\c},\myvec{b\\d}$ are linearly independent then $\vec{N}$ is diagonalizable to $\myvec{0&0\\0&0}$.\\
\begin{align}
\mbox{If }\vec{P}\vec{N}\vec{P}^{-1}=0\\
\mbox{then }\vec{N}=\vec{P}^{-1}\vec{0}\vec{P}=0
\end{align}
So in this case $\vec{N}$ itself is the zero matrix.\\This contradicts the assumption that $\myvec{a\\c},\myvec{b\\d}$
are linearly independent.
Hence we can assume that $\myvec{a\\c},\myvec{b\\d}$ are linearly dependent if both are equal to the zero vector then $\vec{N}$ = 0.Therefore we can assume at least one vector is non-zero.\\
\\
If $\myvec{b\\d}$ is the zero vector then $\vec{N}=\myvec{a&0\\c&0}$\\
\begin{align}
\mbox{So }\vec{N}^2&=0\\
\implies a^2&=0\\
\therefore a&=0\\
\mbox{Thus }\vec{N}&=\myvec{a&0\\c&0}
\end{align}
In this case $\vec{N}$ is similar to $\vec{N}=\myvec{0&0\\1&0}$ via the matrix $\vec{P}=\myvec{c&0\\0&1}$\\
Similary if $\myvec{a\\c}$ is the zero vector,
\begin{align}
\mbox{Then }\vec{N}^2&=0\\
\implies d^2&=0\\
\therefore d&=0\\
\mbox{Thus }\vec{N}&=\myvec{0&b\\0&0}
\end{align}
In this case $\vec{N}$ is similar to $\vec{N}=\myvec{0&0\\b&0}$ via the matrix $\vec{P}=\myvec{0&1\\1&0}$,which is similar to $\myvec{0&0\\1&0}$ as above.
Therefore we can assume neither $\myvec{a\\c}$ or $\myvec{b\\d}$ is the zero vector.\\Since they are linearly dependent we can assume,
\begin{align}
\myvec{b\\d}&=x\myvec{a\\c}\\
\therefore \vec{N}&=\myvec{a&ax\\c&cx}\\
\therefore \vec{N}^2&=0\\
\implies a(a+cx)&=0\\
c(a+cx)&=0\\
ax(a+cx)&=0\\
cx(a+cx)&=0
\end{align}
We know that at least one of a or c is not zero. If a = 0 then since c $\neq$ 0, it must be that x = 0.\\
So in this case $\vec{N}=\myvec{0&0\\c&0}$ which is similar to $\myvec{0&0\\1&0}$ as before.\\
\begin{align}
\mbox{If } a &\neq0\\
\mbox{then }x &\neq0\\
\mbox{else }a(a+cx)&=0\\
\implies a&=0\\
\mbox{Thus }a+cx&=0\\
\mbox{Hence }\vec{N}&=\myvec{a&ax\\ \frac{-a}{x}&-a}
\end{align}
 This is similar to $\myvec{a&a\\-a&-a}$ via $\vec{P}=\myvec{\sqrt{x}&0\\0&\frac{1}{\sqrt{x}}}$. And $\myvec{a&a\\-a&-a}$ is similar to $\myvec{0&0\\-a&0}$ via $\vec{P}=\myvec{-1&-1\\1&0}$ And this finally is similar to $\myvec{0&0\\1&0}$ as before.\\
 Thus either $\vec{N}$ = 0 or $\vec{N}$ is similar over $\mathbb{C}$ to $\myvec{0&0\\1&0}$.
